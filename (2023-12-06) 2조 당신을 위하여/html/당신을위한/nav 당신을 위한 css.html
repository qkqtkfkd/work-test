CSS란

CSS는 웹 사이트에서 화면에 표시되는 정보들을 꾸며주는 역할을 합니다.
CSS는 Cascading Style Sheets의 약자로 HTML, XHML, XML 같은 문서의 스타일을 꾸밀 때 사용하는 스타일 시트 언어입니다.
HTML로 문서의 뼈대를 만들면 CSS는 이 문서를 꾸며 글꼴, 배경색, 위치, 너비와 높이 등을 지정하거나 웹 브라우저, 스크린 크기, 장치에 따라서 화면을 다르게 표시될 수 있도록 지정하는 것입니다.

CSS 선택자

기본 선택자

예시 의미
* { } 전체 요소를 뜻합니다.
p { } p 태그를 뜻합니다.
.container { } container라는 class를 사용하는 태그를 뜻합니다.
#exid { } exid라는 id를 사용하는 태그를 뜻합니다.(한번만 적용)

고급 선택자

예시 의미
section p { } section 요소의 모든 p 요소 선택합니다. (자식의 자식까지 적용)
section > p { } section 요소의 바로 아래 p 요소 선택합니다.
h1 + p { } h1 요소의 바로 다음 형제 p요소 선택합니다.
h1 ~ p { } h1 요소의 모든 형제 p 요소 선택합니다.
a[href] a 요소 중에서 href 속성이 있는 요소 선택합니다.
a[target = _blank] a 요소 중에서 target속성값이 _blank인 요소 선택합니다.
[class ~= button] class 요소 중에서 button 속성값을 포함하는 요소 선택합니다.
a[title |= us] a 요소 중에서 "us" 또는 "us-"로 시작하는 요소 선택합니다.
a[title ^= eng] a 요소 중에서 eng로 시작하는 요소 선택합니다.
[href $= xls] href 속성값이 xls로 끝나는 요소 선택합니다.
[href *= w3] href 속성값이 w3를 포함하는 요소 선택합니다.
