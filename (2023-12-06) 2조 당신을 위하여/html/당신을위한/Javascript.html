JavaScript

Java와 JavaScript의 차이

JavaScript 와 Java는 여러 면에서 비슷하지만 어떤 면에서는 근본적으로 다릅니다. JavaScript 언어는 Java를 닮았지만 Java의 정적 형 지정과 강한 형 검사를 가지고있지 않습니다. JavaScript는 대부분의 Java 표현 구문, 명명 규칙 및 기본적인 흐름 제어 구조를 따르며 이것이 LiveScript에서 JavaScript로 이름이 변경된 이유입니다.

변수

var | let

변수를 선언하고, 선택적으로 초기화할 수 있습니다.

예제1

var x = 1;
if (x === 1) { var x = 2; console.log(x); // 결과값: 2 }
console.log(x); // 결과값: 2

예제 2

let x = 1;
if (x === 1) { let x = 2; console.log(x); // 결과값: 2 }
console.log(x); // 결과값: 1

상수

const

const 선언은 블록 범위의 상수를 선언합니다. 상수의 값은 재할당할 수 없으며 다시 선언할 수도 없습니다.

예제

const number = 42;
console.log(number); // 결과값: 42

Key, Value

예제
