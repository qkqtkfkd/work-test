html이란

HTML은 웹 개발에서 매우 중요한 언어입니다. 파이썬은 웹 개발 분야에서도 사용되기 때문에 HTML을 이해하고 있다면 파이썬 웹 개발에서도 도움이 됩니다. 파이썬 웹 프레임워크에서 HTML 템플릿을 사용하여 웹 애플리케이션을 구축할 수 있습니다. 또한, 파이썬의 BeautifulSoup 라이브러리는 HTML과 XML과 같은 마크업 언어를 파싱하는 데에 사용됩니다. 따라서 HTML을 이해하고 있다면, 파이썬 웹 개발에서 유용하게 사용할 수 있습니다.

사용자로부터 입력을 받을 수 있는 입력 태그

태그 정의 및 특징
input 태그는 사용자로부터 입력을 받을 수 있는 입력 필드(input filed)를 정의할 때 사용합니다
input 요소는 사용자가 데이터를 입력할 수 있는 입력 필드를 선언하기 위해 form 요소 내부에서 사용됩니다.
type= typs 속성의 기본값은 "text"이며 input 요소의
필수 속성은 아니지만 명시하는것이 좋습니다.
type=text 한 줄 텍스트를 입력할수 있는 필드를 정의 합니다.
type=password 비밀 번호를 입력할수 있는 필드를 정의 합니다.
type=search 검색을 입력할수 있는 필드를 정의 합니다.
type=url "URL"은 주소를 입력할수 있는 필드를 정의 합니다.
type=email 이메일 주소를 입력할수 있는 필드를 정의 합니다.
type=tel 전화번호를 만들수 있는 필드를 정의 합니다.
type=checkbox 체크 박스를 만들수 있습니다.
type=radio 서로 연관된 옵션들의 집합을 보통 하나의 라디오 버튼 그룹으로 나타내며
그룹 내의 라디오 버튼은 오직 하나만이 선택될 수 있습니다.
type=number 숫자를 입력할 수 있는 입력 필드를 정의합니다.
type=range 슬라이드 바를 조정하여 범위 내의 숫자를 선택할 수 있는 입력 필드를 정의합니다.
type=date 날짜를 선택할 수 있는 입력 필드를 정의합니다.
입력 결과에는 년도(year), 월(month), 일(day)이 포함되지만, 시간은 포함되지 않습니다.
type=month 날짜를 선택할 수 있는 입력 필드를 정의합니다.
입력 결과에는 년도(year), 월(month)이 포함되지만 일(day), 시간은 포함되지 않습니다.
type=week 날짜를 선택할 수 있는 입력 필드를 정의합니다.
입력 결과에는 년도(year), 주 가 포함되지만 월(month), 일(day), 시간은 포함되지 않습니다.
type=time 시간을 선택할 수 있는 입력 필드를 정의합니다.
이 속성값에는 시(time)와 분(minute)이 포함되며, 오전/오후를 반드시 선택해야만 입력 필드의 유효성 검사를 통과할 수 있습니다.
type=datetime-local 날짜와 시간을 선택할 수 있는 입력 필드를 정의합니다.
이 속성값에는 년도(year), 월(month), 일(day)과 시간이 포함됩니다.
단, 시간을 입력할 때는 오전/오후를 반드시 선택해야만 입력 필드의 유효성 검사를 통과할 수 있습니다.
type=submit 서버의 폼 핸들러(form handler)로 폼 데이터(form date)를 전송하는 제출 버튼(submit button)을 정의합니다.
type=reset 모든 폼(form) 요소의 값을 초깃값으로 되돌리는 리셋 버튼(reset button)을 정의합니다.
type=image 텍스트가 아닌 이미지 형태로 된 제출 버튼을 생성하며, 이때 해당 이미지의 경로는 src 속성에 명시됩니다.
type=button 주로 자바스크립트와 함께 스크립트를 활성화시키기 위해 사용되는 클릭할 수 있는 버튼을 정의합니다.
type=file 업로드할 파일을 선택할 수 있는 입력 필드와 “파일 선택” 버튼을 정의합니다.
type=hidden 사용자에게는 보이지 않는 숨겨진 입력 필드를 정의합니다.

type안에 있는 태그 속성

text, password와 같이 쓰이는 속성들

태그 정의 및 특징
size= size 속성은 input 요소의 너비를 문자수 단위로 명시합니다.
value= text 필드에 보여줄 내용, password에서 사용 안합니다
maxlength= maxlength 속성은 input 요소에 입력할 수 있는 최대 문자수를 명시합니다.

checkbox, radio와 같이 쓰이는 속성들

태그 정의 및 특징
value= value 속성은 체크박스의 value 속성 값을 설정하거나 반환합니다.
checked= 체크박스가 처음부터 체크된 상태로 나타내는 특성입니다
name= radio 전용
여러 옵션의 공통 이름을 나타냅니다.

number, range와 같이 쓰이는 속성들

태그 정의 및 특징
min= 최소값(기본값 0)을 나타냅니다.
max= 최대값(기본값 100)을 나타냅니다.
step= 조정할 단위값(기본값 1)을 나타냅니다.
value= 초기값을 나타냅니다.

submit, reset와 같이 쓰이는 속성들
(이 속성들은 버튼입니다.)

태그 정의 및 특징
value= 버튼에 표시할 내용을 나타냅니다.

button과 같이 쓰이는 속성들

태그 정의 및 특징
value= 버튼에 표시할 내용을 나타냅니다.
onclick= 클릭 시 실행할 JS함수를 나타냅니다.
