    Tag 의 기본

    Document Type Definition : 출력할 웹 페이지 형식을 브라우저에 전달
    문서의 최상단에 위치해야하며, 대소문자를 구분하지 않는다

    HTML Tag : 모든 요소의 부모, 웹 페이지에 단하나만 존재 한다

    title Tag : 웹 페이지의 제목을 정의

    link Tag : HTML 태그에 디자인을 적용하기 위해 사용하는 태그 주로 Css파일을 불러올때 사용

    body Tag : Html 문서 내용을 담당 대부분의 요소들은 body Tag 속에 작성하며 단 하나의 Tag 만 존재
