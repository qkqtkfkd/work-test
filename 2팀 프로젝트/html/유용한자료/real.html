VSCode 이용 시
사용안한 사람은 있어도 한번만 사용한 사람은 없는 단축키

! + Enter, ! + Tab : 도타입 단축키

Ctrl + , : 설정창 열기

Alt + Z : 줄바꿈

Ctrl + Delete : 단어 삭제

Alt + 클릭 : 멀티커서

Alt + 키보드방향키(우아래) : 줄이동

Shift + Delete : 줄 삭제

`(백틱) + Ctrl : 터미널 열기

Ctrl + F : 텍스트 찾기

Ctrl + Shift : 방향키(우or 아래)

Alt + Shift + F : 들여쓰기
